For problem 2:

	To start this program, user can find this in "\Group 7\Problem 2\app_find_k_shortest_path\Find_the_k_shortest_paths.exe"

	1/ user need to type ID of source, destination and k into “Source_text_field”, “Destination_text_field” and “Number_paths_text_field” respectively.

	2/ after filling above information, user need to press the “Find the path” button. After pressing the button, the program will 	run.

	3/ the information of the path (which order nodes and total cost of each paths) will appear in the left hand side of the application. And the line of each path will also appear in the right hand side of the application.


	Moreover, user can zoom the map to show more clearly the path. If user want to show more path which program creates, user can press “the previous path” button to see the previous path or press “the next path” to see the next path.
	
	If  user wants to find more information, the readme file will also useful, it can be reached in “address_folder”.
	
	If you want to check source code, please open the folder "source code" in folder "Problem_2".
